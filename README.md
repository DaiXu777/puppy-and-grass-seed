## 🎮 灵感来源

本作核心玩法灵感来自 **《鲁特里一家死了》**（The Roottrees are Dead）——通过在文档中检索关键词，逐步拼凑出事件全貌的调查推理游戏。

## 🚀 快速游玩（网页版 · 推荐）

**无需安装任何软件，双击即玩：**

1. 下载本项目
2. 打开 `outputs/index.html`
3. 开始调查！

> 或者在 Godot 中打开 `project.godot` 体验引擎版本。

## 📖 剧情背景

你被一个植物系外星文明绑架了。它们看中了你的推理天赋，委托你调查一支失联多年的宇宙探险队——「蒲公英号」。

外星人给了你**五份狗文明档案**。你的任务：按顺序调查五份文档，通过**选中文字→右键搜索**的方式检索关键词，层层推理——

- 探险队员究竟去了哪里？
- 它们为何一代代退化？
- 最终形态的种子，落在了谁的手上？

## 🎮 玩法机制

1. **阅读档案**：每份档案包含大量文本
2. **划词搜索**：选中文字 → 右键 → Search
3. **收集线索**：匹配成功的关键词会解锁线索
4. **搜索记录**：右侧面板记录所有成功搜索
5. **推理进度**：每份档案需找齐线索才能解锁下一份
6. **家族树**：完成档案后填充谱系节点 → 解锁结局
